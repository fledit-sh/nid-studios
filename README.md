# NID Studios

Marketing site for NID Studios AS — a boutique technology consultancy based in Trondheim, Norway. Offers IT infrastructure, graphic design, corporate identity, and penetration testing under one roof.

## Stack

Single self-contained HTML/CSS/JS file. No build step, no dependencies beyond Google Fonts.

## Running

Open `index.html` directly in a browser, or serve the repo root with any static file server:

```
npx serve .
# or
python3 -m http.server
```
