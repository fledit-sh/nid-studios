# CLAUDE.md — NID Studios site

Static marketing site: hand-written HTML + CSS + vanilla JS, no build step, no
backend. Hosted on Vercel; contact form via Formspree. Project-wide security and
privacy guidelines live in the parent `../CLAUDE.md`.

## Security

A security & privacy audit lives in `SECURITY_AUDIT.md`. Always consult it
before changing auth, access control, data handling, or dependencies.
When you fix a finding, update its checkbox and the summary line. Re-run
the full audit when major features or dependencies are added.
